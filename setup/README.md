# Domiknowledge Template

This project was created from the Domiknowledge template. The following
files are **template infrastructure** — they configure Claude Code, the MCP
server, and the Domino plugin. **Do not delete them.**

| File | Purpose |
|------|---------|
| `CLAUDE.md` | Project instructions for Claude Code (read automatically) |
| `.claude/settings.json` | Claude Code plugin configuration |
| `.mcp.json` | MCP server configuration for Domino integration |
| `domino_project_settings.md` | Your Domino project name/owner (edit this if using the MCP server domino_server) |

### Getting Started

1. (Optional for MCP) Edit `domino_project_settings.md` with your project name and owner
2. Start an IDE workspace
3. Open a terminal and run `claude`
4. Try: "What Domino skills do you have?"
